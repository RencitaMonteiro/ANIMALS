Title of the project : VETERINARY APPOINTMENT MANAGEMENT SYSTEM 

DESCRIPTION OF THE PROJECT: A Veterinarian is a professional who treates the injuries and illness of animals. We use medical tools and X-rays machine for the same. In                               some situations we perform emergency surgeries. It is our goal for your pets to live long and healthy lives, and we offer friendly and                                   personalized attention to make sure this happens. No matter what type of treatment your animal is receiving, you can trust that they will be                             comfortable in our state-of-the-art, inviting facility.

Tech stack used: a) Front-End :- HTML,CSS
                 b)Back-End :- My Sql
                 c) Scripting language :- PHP
                 
How to run the Doctor Appointment Management System Project Using PHP and MySQL
   1. Download the zip file
   2. Extract the file and copy dams folder
   3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/HTML)
   4.Open PHPMyAdmin (http://localhost/phpmyadmin)
   5. Create a database with the name damsmsdb
   6. Import damsmsdb.sql file(given inside the zip package in the SQL file folder)
   7. Run the script http://localhost/dams
